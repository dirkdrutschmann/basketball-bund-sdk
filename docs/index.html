Installation

Das Basketball Bund SDK kann über npm installiert werden:

npm install basketball-bund-sdk
Hinweis: Das SDK unterstützt sowohl ES Modules als auch CommonJS.

Quick Start

ES Modules (empfohlen)

import { BasketballBundSDK } from 'basketball-bund-sdk';

const sdk = new BasketballBundSDK({
    baseUrl: 'https://www.basketball-bund.net/rest',
    username: 'your-username',
    password: 'your-password'
});

// Automatische Authentifizierung
await sdk.authenticate('your-username', 'your-password');

// Liga-Liste abrufen
const ligaList = await sdk.wam.getLigaList({
    akgGeschlechtIds: ['1'],
    altersklasseIds: [1],
    gebietIds: [1],
    ligatypIds: [1],
    sortBy: 1,
    spielklasseIds: [1],
    token: 'your-token',
    verbandIds: [1]
});

console.log(ligaList);

CommonJS

const sdk = new BasketballBundSDK({
    baseUrl: 'https://www.basketball-bund.net/rest'
});

// Authentifizierung

console.log('Login erfolgreich:', isAuthenticated);

Authentifizierung

Das SDK unterstützt automatische Authentifizierung über SESSION-Cookies:

authenticate(username: string, password: string): Promise<boolean>

Führt eine Authentifizierung durch und gibt true zurück, wenn erfolgreich.

isAuthenticated(): Promise<boolean>

Überprüft den aktuellen Authentifizierungsstatus durch einen API-Aufruf an /user/lc.

Authentifizierungsbeispiel
const sdk = new BasketballBundSDK();

// Login durchführen
const loginSuccess = await sdk.authenticate('username', 'password');
if (loginSuccess) {
    console.log('Login erfolgreich');
    
    // Status überprüfen
    const isLoggedIn = await sdk.isAuthenticated();
    console.log('Eingeloggt:', isLoggedIn);
} else {
    console.log('Login fehlgeschlagen');
}

BasketballBundSDK

Die Hauptklasse des SDK. Verwaltet alle Services und die HTTP-Kommunikation.

Konstruktor

new BasketballBundSDK(config?: BasketballBundSDKConfig)

Eigenschaften

Eigenschaft Typ Beschreibung
auth AuthenticationService Service für Authentifizierung
wam WamService Service für WAM-Funktionen
club ClubService Service für Vereins-Funktionen
match MatchService Service für Spiel-Funktionen
competition CompetitionService Service für Wettbewerbs-Funktionen
team TeamService Service für Team-Funktionen
user UserService Service für Benutzer-Funktionen
registration RegistrationService Service für Registrierungs-Funktionen
captcha CaptchaService Service für Captcha-Funktionen

Methoden

authenticate(username: string, password: string): Promise<boolean>

Führt eine Authentifizierung durch.

isAuthenticated(): Promise<boolean>

Überprüft den Authentifizierungsstatus.

setHeaders(headers: Record<string, string>): void

Setzt HTTP-Header für alle nachfolgenden Anfragen.

setBaseUrl(url: string): void

Ändert die Basis-URL für API-Anfragen.

Response-Typen

Das SDK gibt direkt die Daten aus der API zurück, ohne Wrapper-Objekte.

Wichtig: Alle Service-Methoden geben direkt die Daten zurück (z.B. LigaList statt Response<LigaList>).

Beispiel

// Vorher (mit Response-Wrapper)
const response = await sdk.wam.getLigaList(params);
const ligaList = response.data;

// Jetzt (direkte Daten-Rückgabe)
const ligaList = await sdk.wam.getLigaList(params);
// ligaList ist direkt vom Typ LigaList

Grundlegende Nutzung

SDK initialisieren und Liga-Liste abrufen
import { BasketballBundSDK } from 'basketball-bund-sdk';

const sdk = new BasketballBundSDK({
    baseUrl: 'https://www.basketball-bund.net/rest'
});

// Authentifizierung
await sdk.authenticate('username', 'password');

// Liga-Liste abrufen
const ligaList = await sdk.wam.getLigaList({
    akgGeschlechtIds: ['1'],
    altersklasseIds: [1],
    gebietIds: [1],
    ligatypIds: [1],
    sortBy: 1,
    spielklasseIds: [1],
    token: 'your-token',
    verbandIds: [1]
});

console.log('Gefundene Ligen:', ligaList.ligen.length);
console.log('Weitere Daten verfügbar:', ligaList.hasMoreData);
Vereinssuche durchführen
// Vereine nach Namen suchen
const clubResults = await sdk.club.searchClub({
    name: 'Basketball',
    plz: '12345'
});

console.log('Gefundene Vereine:', clubResults.clubs.length);

// Details zu einem Verein abrufen
if (clubResults.clubs.length > 0) {
    const clubDetails = await sdk.club.getClubDetails({
        clubId: clubResults.clubs[0].id
    });
    console.log('Vereinsname:', clubDetails.name);
}

Authentifizierungsbeispiele

Automatische Authentifizierung beim Initialisieren
const sdk = new BasketballBundSDK({
    username: 'your-username',
    password: 'your-password'
});

// SDK ist automatisch authentifiziert
const isLoggedIn = await sdk.isAuthenticated();
console.log('Eingeloggt:', isLoggedIn);
Manuelle Authentifizierung
const sdk = new BasketballBundSDK();

// Login durchführen
const loginSuccess = await sdk.authenticate('username', 'password');
if (loginSuccess) {
    console.log('Login erfolgreich');
    
    // Status überprüfen
    const isLoggedIn = await sdk.isAuthenticated();
    console.log('Eingeloggt:', isLoggedIn);
    
    // Benutzerkontext abrufen
    const userContext = await sdk.user.getUserContext();
    console.log('Benutzer:', userContext.username);
} else {
    console.log('Login fehlgeschlagen');
}
Logout durchführen
// Benutzer abmelden
await sdk.auth.logout();

// Status überprüfen
const isLoggedIn = await sdk.isAuthenticated();
console.log('Eingeloggt:', isLoggedIn); // false

Captcha für Registrierung verwenden
// Captcha generieren
const captcha = await sdk.captcha.generate();
console.log('Captcha Code:', captcha.captchaCode);

// Benutzer registrieren
const registerResult = await sdk.registration.registerUser({
    username: 'newuser',
    email: 'user@example.com',
    password: 'securepassword',
    captchaCode: captcha.captchaCode,
    captchaId: captcha.id
});

if (registerResult.success) {
    console.log('Registrierung erfolgreich');
}
Fehlerbehandlung
try {
    const ligaList = await sdk.wam.getLigaList(params);
    console.log('Erfolgreich abgerufen');
} catch (error) {
    if (error.status === 401) {
        console.log('Nicht authentifiziert - Login erforderlich');
        await sdk.authenticate('username', 'password');
        // Erneut versuchen
        const ligaList = await sdk.wam.getLigaList(params);
    } else if (error.status === 404) {
        console.log('Ressource nicht gefunden');
    } else {
        console.log('Fehler:', error.message);
    }
}
